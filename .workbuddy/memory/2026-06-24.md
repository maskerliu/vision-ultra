### 2026-06-24 人脸比对识别功能实现

#### 完成的工作
1. **ProcessorManager.ts** 新增 `faceRecognize()` 方法：
   - 从当前人脸 landmarks 提取特征向量
   - 通过 `FaceRec.recognize()` 调用后端 LanceDB 相似度搜索
   - 节流控制：每 800ms 最多识别一次，避免频繁请求
   - 支持 `enableRecognize` 开关控制

2. **WorkerManager.ts** `onFaceDetectMsg` 中：
   - 人脸有效时自动触发 `faceRecognize()`
   - 人脸消失时自动调用 `clearRecognizeResult()` 清空结果

3. **DetectRec.vue** UI 改造：
   - 识别开关改为 `van-checkbox`（contact icon）
   - 识别结果从底部卡片移入顶部栏（紧凑展示：头像 + 人名 + 相似度%）
   - 未识别时显示 "识别中" loading 提示
   - 移除底部 `recognize-card` 和 `no-recognize-hint`
   - 新增 `.rec-checkbox`、`.rec-top-bar` 等紧凑样式

4. **状态同步**：
   - `onRecognizeChange` 回调同步 `processorMgr.enableRecognize`
   - `initProcessorMgr` 初始化时也同步 `enableRecognize` 状态

5. **类型检查**：tsc --noEmit 无新增错误（仅既有 App.vue 模块声明错误）
